How I became a self-taught developer through hands-on project building.

2023 - Early Days
Started Coding Journey
Began learning programming through online resources and tutorials. Started with Python basics and quickly moved to building small automation scripts. Discovered the power of problem-solving through code.
2023 - Mid Year
First Computer Vision Project
Built a gesture recognition system using OpenCV and machine learning. Learned about image processing, feature detection, and real-time processing. This project sparked my interest in practical AI applications.
2024 - Q1
Discovered Android Development
Started learning Kotlin and Android Studio. Fell in love with mobile development and the challenge of creating user-facing applications. Learned about Material Design, app architecture, and the Android ecosystem.
2024 - Q2
Built Face Attendance System
Created a complete attendance tracking system with face recognition. Integrated FastAPI backend with Python, PostgreSQL database via Supabase, and built an admin dashboard. Learned full-stack development patterns.

Created a full-featured OTT streaming application using Kotlin. Implemented custom ExoPlayer integration, user authentication with Supabase, watchlists, and Material Design 3 UI. This became my flagship Android project.
2024 - Q4
Explored Web Development
Expanded into web technologies - React, TypeScript, and modern CSS. Built e-commerce platforms and agency websites. Learned responsive design, performance optimization, and SEO best practices.
2025 - Present
